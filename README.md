# Waffle
Waffle core system

## Building Waffle

Gradle is a java building tool. To build, simply run

```bash
# for mac/linux/cygwin(?)
$ gradlew build
# for windows (haven't tried)
$ gradlew.bat build
```

To run,

```bash
$ gradlew run
```

OR expand zip file in `build/distriputions/waffle.zip` and run `waffle` or `waffle.bat`.
